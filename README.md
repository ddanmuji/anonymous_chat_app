## ✍🏼 lesson learned

### 유니캐스팅 (emit & on)

#### emit

- 송신 이벤트 발생
- emit은 소켓을 통해서 이벤트를 보낼 때 사용이 된다.

#### on

- 수신 이벤트 등록
- emit으로 발생한 이벤트를 on으로 받을 수 있다.
